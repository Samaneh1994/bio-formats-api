# Add new metadata entity and property definitions to this file.

# Path key and syntax notes:
# 1) Entities ending in plus (+) have an associated counter method.
# 2) Elements ending in plus (+) have maxOccurs > 1 and generate an index
#    parameter to allow differentiation between them.
# 3) Elements beginning with at (@) are not direct children of the previous
#    element, but instead reference the parent via an ID reference.
# 4) Elements beginning with at-bang (@!) also use a parent reference but
#    the API is slightly different -- e.g., for a LightSource accessing a
#    child Laser, the method is lightSource.getLaserListByLightSource()
#    rather than lightSource.getLaserList().
# 5) Overridden attributes ending in percent (%) indicate a type conflict
#    that must be resolved by calling type conversion methods from the 
#    OMEXMLMetadata superclass.
# 6) Attributes starting with small case indicate a method prefix other than
#    "get" -- e.g., "isTunable" overrides default "getTunable" method name.

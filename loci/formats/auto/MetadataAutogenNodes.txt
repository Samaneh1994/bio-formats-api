[Dimensions]
a set of pixel dimensions in the metadata store with a particular index. Unless both values are non-null, the MetadataStore should assume pixelSizeX equals pixelSizeY (i.e., should populate the null field with the other field's value)
2004: Image+/CustomAttributes/Dimensions
200706: Image+
Float PixelSizeX  size of an individual pixel's X axis in microns
Float PixelSizeY  size of an individual pixel's Y axis in microns
Float PixelSizeZ  size of an individual pixel's Z axis in microns
Float PixelSizeC  distance between adjacent wavelengths in nanometers
Float PixelSizeT  distance between adjacent time points in seconds

Integer X0             the starting X coordinate
Integer Y0             the starting Y coordinate
Integer Z0             the starting Z coordinate
Integer X1             the ending X coordinate
Integer Y1             the ending Y coordinate
Integer Z1             the ending Z coordinate
Integer T0             the starting timepoint
Integer T1             the ending timepoint
Object DisplayOptions  the display options to attach to this region of interest

= LIMReader =

This page lists supported metadata fields for the
Bio-Formats Laboratory Imaging format reader.

These fields are from the [http://www.ome-xml.org/ OME data model].
Bio-Formats standardizes each format's original metadata to and from the OME
data model so that you can work with a particular piece of metadata (e.g.,
physical width of the image in microns) in a format-independent way.

Of the 313 fields documented below:
  * The file format itself supports 15 of them (4%).
  * Of those, Bio-Formats fully or partially converts 15 (100%).

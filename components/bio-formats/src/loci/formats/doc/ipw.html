File format name: Image-Pro Workspace
Maintainer: Media Cybernetics
Software that produces this:Image-Pro Plus
Other software that reads this format:Image-Pro Plus
Specifications: none :-(
Structure:
IPW files are actually a variant of Microsoft's OLE 2 document format. They contain embedded TIFF files, along with minimal metadata needed to read the images. The IPW format does support storing multiple datasets within the same file; however, this is not implemented in Bio-Formats (if you have an IPW file with multiple datasets, please send it to us and we will add support for it).

Note that you will need to install Jakarta POI in order to read IPW files.
